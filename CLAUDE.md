# DATA 111 Fall 2026 — Student Repo Notes

Student-facing materials for DATA 111 (Intro to Data Science, Cal Poly Humboldt). **Full project standards live in `../data111_fall26_private/CLAUDE.md` — read that first.**

## Rules for this folder

- Everything here is visible to students: **no assignment solutions, no test internals, no grading files.** (Exception: `midterm/` and `final practice/` intentionally include practice solutions.)
- Notebooks here are **generated** from Otter Assign masters in `../data111_fall26_private/raw_notebooks/`. Don't hand-edit them — edit the master and regenerate.
- **Never edit anything under `hw/` or `lab/` without Emily's explicit approval — the autograder depends on them.**
- This repo publishes to **https://github.com/mlekimchi/data111_fall26**; type-along links in demo notes point there.
- Layout: `hw/hwNN/hwNN.ipynb`, `lab/labNN/labNN.ipynb`, `project/`, `midterm/`, `final practice/`. Each assignment folder is self-contained (data files, images, `d8error.py`, `errorConfig.json` alongside the notebook).
- `lectures/` holds demo notes (`lecNN.ipynb` = complete, `lecNN_live.ipynb` = blanked code-along) plus shared datasets/images, all loose (not per-lecture folders). Restyled ekc versions get new filenames (`lecNN-ekc.ipynb`); originals stay untouched.
- `CPH_academic calendar 2025-2026.pdf` — check before scheduling anything.
- Update `README.md` if the repo's purpose or term changes (currently says Fall 2025 — update to Fall 2026 when publishing).
